This package is adapted from collective/icalendar
(https://github.com/collective/icalendar), commit f92bd57 (version 7.2.2),
BSD-2-Clause (see LICENSES/icalendar-LICENSE in the repository root).

It serves as the vobject-rs distribution's icalendar compatibility layer;
parser internals are progressively delegated to the Rust core
(vobject._core). The upstream test suite ships in icalendar/tests and runs
against this copy.

Local modifications to the vendored copy:
- _version.py: static version stamp (upstream generates it via hatch-vcs).
- tests/test_with_doctest.py: documentation doctests point at the vendored
  upstream_docs/ tree instead of the repository root.
- tests/test_funding_json.py: replaced with a module-level skip; it checks
  the upstream repository's funding manifest, not library behavior.
- The repository provides a src/icalendar/tests symlink so that upstream
  doctests which open fixtures via the upstream repo-relative path
  (src/icalendar/tests/...) keep working from this repository's root.
